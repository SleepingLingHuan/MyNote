# MyNote

MyNote 是一个本地优先的日记、随笔和个人写作工具。

它面向那些希望长期保存私人文字的人：日记、杂谈、学习记录、读书笔记、情绪片段、生活整理，都可以放在一个安静、轻量、没有账号体系的空间里。MyNote 不追求社交分享，也不把写作变成复杂的管理流程；它更像一张长期摆在手边的个人书桌。

你可以把它当作一个本地个人博客，也可以把它当作每天打开的私人笔记本。所有内容都围绕“文章”组织，每篇文章只属于一个分类，避免标签过多带来的整理负担。

## 主要特性

- 本地写作：适合日记、随笔、杂谈、学习记录和长期私人文本积累。
- 分类整理：每篇文章归属一个分类，也支持子分类，结构清晰。
- Markdown 写作：用熟悉的 Markdown 方式记录内容。
- 图片插入：文章可以保存图片内容，适合图文记录。
- 编辑与预览：在编辑和预览状态之间快速切换。
- 本地保存：内容保存在本地，不依赖账号和云端服务。
- 导入导出：支持将内容导出备份，也可以重新导入。
- 浅色与深色主题：根据使用环境切换不同视觉风格。
- 桌面端使用：适合作为日常打开的本地写作应用。

## 安装

### macOS

1. 前往项目的 Releases 页面，下载最新版本的 `MyNote_*.dmg` 文件。
2. 打开下载好的 DMG 文件。
3. 将 `MyNote.app` 拖入 `Applications` 文件夹。
4. 在「应用程序」中打开 MyNote。

如果 macOS 提示应用来自未知开发者，可以在「系统设置」中允许打开，或右键点击 `MyNote.app` 后选择「打开」。

当前发布包优先面向 Apple Silicon Mac。如果你使用的是 Intel Mac，请留意 Releases 页面是否提供对应版本。

## 设计理念

MyNote 希望保持清新、简约、克制的视觉风格。界面尽量减少干扰，把注意力留给文字本身。

它不是团队协作工具，也不是复杂知识库系统。它更关注一个人的长期写作体验：打开、记录、保存、回看。

## 适合谁

- 想要一个本地日记工具的人
- 喜欢 Markdown 写作的人
- 希望用分类管理个人文章的人
- 不想依赖账号、网络或云服务的人
- 想长期维护私人随笔和生活记录的人
